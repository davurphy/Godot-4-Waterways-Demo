# Godot Debug Logger

A Godot 4 addon that writes all debug output to a text file in your project folder.

## Installation

1. Copy this folder into your project's `addons` folder
2. Enable the plugin: Project → Project Settings → Plugins
3. Add runtime_logger.gd as Autoload: Project → Project Settings → Autoload
   - Path: res://addons/godot_debug_logger/runtime_logger.gd
   - Name: GodotDebugLogger

## Output

Logs go to: `your_project/debugger/godot_debug.txt`

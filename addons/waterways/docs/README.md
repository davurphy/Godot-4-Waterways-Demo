# Waterways Documentation

This folder collects the working documentation for the Waterways add-on.

## Folders

- `history/`: chronological change history.
- `handoffs/`: current and future session handoffs.
- `roadmaps/`: long-running planning documents and review workflows.
- `research/`: shared citation and source indexes.
- `audit/`: focused audits and technical reviews.
- `ideas/`: possible future directions that are not yet scoped work.
- `spec-driven/`: feature-spec workflow, templates, and per-feature planning folders.
